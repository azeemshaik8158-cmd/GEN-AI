# AI Research & Decision Assistant

## Project Overview
This project is focused on building an AI-powered research and decision support assistant.  
It helps in understanding documents, extracting meaningful insights, identifying important patterns, and generating helpful recommendations using Large Language Models (LLMs).

The main goal is to support structured reasoning instead of just giving raw answers.
